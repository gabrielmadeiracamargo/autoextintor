
# Sistema Autônomo de Combate a Incêndios: S.A.C.I

O S.A.C.I tem o poder de apagar fogo no exato ponto de início usando visão computacional pelo main.py, e controlando motores para apontar uma mangueira e ativar a bomba pelo electronics.ino.

Foram usados:

+ 1x Webcam
+ 1x Arduino (Nano);
+ 2x servomotores MG995;
+ 1x bomba d'água submersa 3-6V;
+ 2x fontes 5V 3A (uma pros motores e outra pra bomba, a fim de evitar que picos de energia simultâneos consumam mais corrente do que o provido por apenas uma fonte);
+ 1x Transistor TIP122 (para permitir ativar e desativar a bomba pelo código do Arduino);
+ Diodo, resistor 880Ω e alguns jumpers.

Para montagem da caixa:
+ 1x Suporte pan/tilt pros servos impresso 3D;
+ 1x Encaixe pra mangueira no suporte impressa 3D;
+ 1x Bico pra mangueira aumentar a pressão impresso 3D;
+ Uma caixa de madeira aberta na frente e furada atrás para passar fios. Suporte é parafusado nela
